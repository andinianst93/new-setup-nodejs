# How to run?

```bash
npm install

npm run watch

npm run dev

# production
npm run start
```

```bash
# env example
MONGO_INITDB_ROOT_USERNAME="username"
MONGO_INITDB_ROOT_PASSWORD="password"
MONGO_URI="mongodb://username:password@domain:port/dbname?authSource=admin"
JWT_KEY="example_key"
NODE_ENV="production"
```
